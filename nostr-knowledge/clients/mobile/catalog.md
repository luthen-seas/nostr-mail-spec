# Mobile Client Catalog

> Comprehensive catalog of Nostr mobile clients for iOS, Android, and cross-platform.

---

## iOS

| Name | URL | Language | Description |
|------|-----|----------|-------------|
| **Damus** | [damus-io/damus](https://github.com/damus-io/damus) | Swift | THE flagship iOS client. Twitter-like feed with zaps. Drove early Nostr adoption. By Will Casarin. |
| **Nos** | [planetary-social/nos](https://github.com/planetary-social/nos) | Swift | UX-focused client with safety features. By the former Scuttlebutt/Planetary team. |
| **Nostur** | [nostur.com](https://nostur.com) | Swift | Full-featured iOS and macOS client with extensive NIP support. |
| **Nooti** | [nootti.com](https://nootti.com) | Swift | Cross-posting iOS and iPad client. Publish to Nostr and other platforms. |
| **Tamga** | [erdaltoprak/tamga](https://github.com/erdaltoprak/tamga) | Swift | Offline-first contact and profile manager. |
| **Pika** | [sledtools/pika](https://github.com/sledtools/pika) | Swift | End-to-end encrypted messaging. |

## Android

| Name | URL | Language | Description |
|------|-----|----------|-------------|

| **Voyage** | [dluvian/voyage](https://github.com/dluvian/voyage) | Kotlin | Reddit-like UI for topic browsing and threaded discussions. |

| **Spring Browser** | [spring.site](https://spring.site) | Kotlin | Nostr-focused mobile browser. |
| **Zemzeme** | [whisperbit-labs/zemzeme-android](https://github.com/whisperbit-labs/zemzeme-android) | Kotlin | Private messaging focused. |

## Cross-Platform (iOS + Android)

| Name | URL | Language | Description |
|------|-----|----------|-------------|
| **Primal** | [primal.net](https://primal.net) | Various | iOS, Android, and web. Built-in Lightning wallet. Caching CDN for speed. |
| **0xchat** | [0xchat-app](https://github.com/0xchat-app) | Dart/Flutter | Telegram-like messaging for iOS and Android. E2E encrypted. |
| **Nostrmo** | [haorendashu/nostrmo](https://github.com/haorendashu/nostrmo) | Dart/Flutter | Multi-platform Flutter client covering iOS, Android, web, and desktop. |
| **YakiHonne** | [yakihonne.com](https://yakihonne.com) | Various | Multi-format content protocol. iOS and Android. |
| **Nostrid** | [lapulpeta/Nostrid](https://github.com/lapulpeta/Nostrid) | C#/MAUI | Cross-platform via .NET MAUI: Android, Windows, macOS, Linux. |

| **Keychat** | [keychat-io/keychat-app](https://github.com/keychat-io/keychat-app) | Dart/Flutter | Signal-protocol encrypted chat with Bitcoin key management. |
